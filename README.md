# Market-Simulator

Market simulator with multiple assets and markets, and cooperating companies, investors and funds.

## Installation
```
git clone https://github.com/Shapke01/Market-Simulator.git
```
```
cd Market-Simulator
```

## Technologies
* java 17
* Maven
* JFreeChart
* Swing
